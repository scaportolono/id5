# IdentityV スポーン位置覚え丸
スポーンパターンをサバイバー、ハンター視点で覚えたりするためのWebアプリです。

# Requirement

- Node.js v14 (14.3.0)
- Webpack v4 (4.39.3)

# Setup

```bash
git clone https://github.com/scaportolono/id5.git
# またはダウンロード

cd id5

npm install

npm start
# http://localhost:8080/ で起動
```

# Development

```bash
wabpack
```
